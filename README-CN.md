## 项目介绍

<!-- PROJECT SCREENSHOT -->

通过随机菜单 Web 应用来决定今天吃什么吧！本项目由 HTML/CSS/ECMAScript 和 Claude3 Haiku 生成的代码进行拼接而开发完成，零依赖且数据存储至本地，仅由一个 HTML 文件组成。
